# 3D射击游戏原型项目

## 项目概述
这是一个基于Unity的简单3D射击游戏原型，包含：
- 第一人称视角
- 基本射击功能
- 简单敌人AI
- 场景交互

## 技术栈
- Unity 2022+ (推荐2023.2)
- C#编程语言
- Unity内置组件系统

## 文件结构
- Assets/Scripts/ - C#脚本文件
- Assets/Scenes/ - 场景文件
- Assets/Models/ - 模型和材质
- Assets/Prefabs/ - 预制体

## 开发步骤
1. 创建Unity项目
2. 设置第一人称控制器
3. 添加射击系统
4. 创建敌人AI
5. 设计简单场景
6. 添加音效和UI
